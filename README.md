# 📊 India Multi-Indicator Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing multiple health indicators across Indian states using NDAP dataset. The goal is to identify trends, patterns, and disparities in health conditions.

## 🛠 Tools & Technologies

* Power BI
* Power Query
* Excel / CSV

## 📊 Key Features

* KPI cards showing key health indicators (Anaemia, Diabetes, Hypertension, Diarrhoea)
* State-wise comparison using heatmap table
* Interactive map visualization of disease distribution
* Filter options for State, Disease, and Group
* Comparison of rural vs urban health metrics

## 📈 Key Insights

* Anaemia levels are significantly higher in several states indicating public health concern
* Rural areas show higher average values for multiple diseases compared to urban areas
* Certain states consistently show higher disease prevalence across indicators
* Variation in cancer screening suggests uneven healthcare accessibility
